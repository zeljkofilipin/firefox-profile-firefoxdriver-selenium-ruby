# Firefox profile + FirefoxDriver + Selenium + Ruby

## Prerequisites

- [Firefox 47.0.1](https://ftp.mozilla.org/pub/firefox/releases/47.0.1/)

## Installation

    gem install bundler
    bundle install

## Usage

    bundle exec ruby firefox.rb
